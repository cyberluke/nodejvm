eval

open static fun eval(js: String!): Value!

Evaluates the given string and returns a generic Value object representing the result, that can then be converted into more useful forms. Note that you must be on the NodeJS thread for this to work (see #runJS(Supplier) for how to do this).

Exceptions

IllegalStateException - if you're not on the NodeJS thread.
