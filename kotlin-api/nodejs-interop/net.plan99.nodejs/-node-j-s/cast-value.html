castValue

open static fun <T : Any!> castValue(value: Value!, typeLiteral: TypeLiteral<T>!): T

Converts the Value to a JVM type in the following way:

  1. If the type is an interface not annotated with @FunctionalInterface then a special proxy is returned that knows how to map JavaBean style property methods on that interface to JavaScript properties.
  2. Otherwise, the Value#as method is used with the TypeLiteral so generics are preserved and the best possible translation occurs.
